# C4FAFG5F28KHN

Markdown Note Editor — un éditeur de notes minimaliste avec rendu Markdown en temps réel.

## Concept

Un éditeur de notes léger qui convertit le Markdown en HTML instantanément. Sauvegarde locale via LocalStorage, interface épurée.

## Fonctionnalités

- Édition Markdown en temps réel
- Prévisualisation instantanée
- Sauvegarde automatique (LocalStorage)
- Export en HTML
- Mode focus (plein écran)

## Technologies

- HTML5, CSS3, JavaScript vanilla
- Aucun framework, aucune dépendance

## Déploiement

```bash
git clone https://github.com/b29w02m4nf8/C4FAFG5F28KHN.git
cd C4FAFG5F28KHN
# Ouvrir index.html dans le navigateur
```
